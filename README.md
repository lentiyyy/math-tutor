## Описание проекта

В рамках итогового проекта разработан многостраничный адаптивный веб-сайт для репетитора по математике.

Сайт предназначен для презентации услуг преподавателя, ознакомления пользователей с условиями обучения, отзывами учеников и возможностью записи на занятие через форму обратной связи.

Проект реализован в формате статического веб-приложения с использованием HTML, CSS и JavaScript.

---

## Основной функционал

- Главная страница с презентацией услуг;
- Страница с тарифами и описанием занятий;
- Страница с отзывами учеников;
- Контактная страница с формой записи;
- Адаптивная верстка для мобильных устройств;
- Переключение светлой и тёмной темы;
- Мобильное меню;
- Динамическая подгрузка дополнительных отзывов;
- Валидация формы записи.

---

## Используемые технологии
